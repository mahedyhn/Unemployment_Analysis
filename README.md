# Unemployment Analysis with Python

This project was completed as part of the **CodeAlpha Internship**. The goal is to analyze the unemployment rate in India, focusing on the impact of the COVID-19 pandemic.

## 📌 Project Overview
- **Objective:** Data cleaning, exploration, and visualization of unemployment trends.
- **Tools Used:** Python (Pandas, Matplotlib, Seaborn, Plotly).
- **Key Analysis:** Impact of the 2020 lockdown on regional and area-wise unemployment rates.

## 📊 Key Insights
- Significant spike in unemployment during April-May 2020 across all states.
- High disparity between Urban and Rural unemployment rates during the lockdown.
- States like Bihar and Jharkhand showed the highest peak unemployment rates.

## 🚀 How to Run
1. Clone the repository.
2. Install dependencies: `pip install pandas matplotlib seaborn plotly`.
3. Run the script: `python analysis.py`.
